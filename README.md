# 構造型タイムラインモデル
### Structural Timeline Model

---

## 概要

このリポジトリは、**Hiroko** が考案した「構造型タイムラインモデル」の説明資料です。

「前世」や「タイムライン」を、時間の直線としてではなく、**構造**として理解するモデルです。
同じ構造の中に、複数の人格・人生が同時に存在するという視点から組み立てられています。

---

## モデルの概要

一般的なスピリチュアルでは、人生はこのように説明されます。

```
過去 → 現在 → 未来
```

このモデルでは、それを構造として置き換えます。

| 概念 | 定義 |
|---|---|
| 魂 | 木 |
| 人格 | 葉（複数、同列に存在） |
| 圧力点 | パターン解放が起きるポイント |
| 接続点 | 別人格の情報を既知感として受け取るポイント |

人格同士は時間の前後関係ではなく、**同じ構造内に同時存在**しています。
どの人格から見るかによって「窓口」が変わります。
